1. What is JSX, and why is it used?
   Answer: JSX (JavaScript XML) হলো React–এ ব্যবহৃত একটি বিশেষ সিনট্যাক্স, যা JavaScript কোডের ভেতরে HTML-এর মতো কোড লেখার সুবিধা দেয়।
   1. কোড সহজ ও পড়তে সুবিধাজনক
   2. HTML ও JavaScript একসাথে লেখা যায়
   3. ডায়নামিক ডাটা ব্যবহার করা যায়

2. What is the difference between State and Props?
   Answer: Props = বাইরে থেকে পাওয়া ডেটা
           State = নিজের ভিতরের ডেটা

3. What is the useState hook, and how does it work?
   Answer: useState হলো এমন একটি Hook যা Function Component–এ State যোগ করে এবং State পরিবর্তন হলে UI স্বয়ংক্রিয়ভাবে আপডেট করে।

4. How can you share state between components in React?
   Answer: State উপরে তুলুন (Lift it up)
           অথবা Context/Global Store ব্যবহার করুন

5. How is event handling done in React?
   Answer: camelCase ইভেন্ট নাম ব্যবহার করে
           Function দিয়ে handler তৈরি করে
           JSX-এর ভিতরে {} ব্যবহার করে
